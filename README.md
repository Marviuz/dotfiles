# dotfiles

Powershell:

```
New-Item -ItemType SymbolicLink -Path "C:\Users\<USER>\AppData\Local\<DIRECTORY>" -Target "E:\dotfiles\<DIRECTORY>"
```

_Note: Also create symlink for `nvim-data`_
